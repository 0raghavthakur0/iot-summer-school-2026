# Traffic Light Controller

## Components Required

- Arduino UNO
- Red LED
- Yellow LED
- Green LED
- Push Button
- 220Ω Resistors
- Breadboard
- Jumper Wires

---

## Pin Connections

Red LED → Pin 2

Yellow LED → Pin 3

Green LED → Pin 4

Push Button → Pin 7

---

## Working

- Red LED glows for 5 seconds.
- Yellow LED glows for 2 seconds.
- Green LED glows for 4 seconds.
- Pressing the pedestrian button immediately turns the Red LED ON for 8 seconds.

---

## Expected Output

Serial Monitor displays:

```
1234 ms : RED
6234 ms : YELLOW
8234 ms : GREEN
12345 ms : PEDESTRIAN CROSSING
```
