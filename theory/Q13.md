# Q13 – Arduino UNO Pin Types

## 1. Digital Input Pin

### Description
A Digital Input pin is used to read only two states:
- HIGH (1)
- LOW (0)

### Example
Push Button connected to Pin 2.

### IoT Use Case
Detecting whether a door is open or closed using a magnetic switch.

---

## 2. Digital Output Pin

### Description
A Digital Output pin sends HIGH or LOW signals to control external devices.

### Example
LED connected to Pin 13.

### IoT Use Case
Turning ON/OFF a relay to control home appliances.

---

## 3. Analog Input Pin

### Description
Analog Input pins read variable voltages between 0V and 5V.

Arduino converts the voltage into values from 0–1023 using its ADC.

### Example
Potentiometer connected to A0.

### IoT Use Case
Reading temperature, light intensity, or soil moisture.

---

## 4. PWM Output Pin

### Description
PWM (Pulse Width Modulation) pins simulate analog output by rapidly switching ON and OFF.

PWM Pins on Arduino UNO:
3, 5, 6, 9, 10, 11

### Example
LED brightness control.

### IoT Use Case
Controlling motor speed or dimming LED lights.

---

## 5. I2C / SPI Pins

### I2C

Used for communication between Arduino and multiple devices using only two wires.

Pins:
- SDA → A4
- SCL → A5

### Example
LCD Display

### IoT Use Case
Connecting temperature sensors or OLED displays.

---

### SPI

Used for high-speed communication.

Pins:
- MOSI → 11
- MISO → 12
- SCK → 13
- SS → 10

### Example
SD Card Module

### IoT Use Case
Data logging and wireless communication modules.

---

Arduino UNO provides different pin types for reading sensors, controlling actuators, generating PWM signals, and communicating with IoT devices, making it suitable for a wide variety of embedded and IoT applications.
