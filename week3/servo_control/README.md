# Servo Motor Sweep

## Components

- Arduino UNO
- SG90 Servo Motor
- 10kΩ Potentiometer
- Push Button
- Breadboard
- Jumper Wires

## Connections

Servo Signal → Pin 9

Potentiometer → A0

Push Button → Pin 2

## Features

- Potentiometer controls servo angle (0°–180°).
- Current angle displayed on Serial Monitor.
- Button performs one complete sweep (0° → 180° → 0°).
